O que é o BATOCERA-HD-EDITION?
--------------------------------------------------------------------------------
    Trata-se de um adicional que permite rodar o Batocera diretamente do HD sem
usar Pendrive, sem adicionar HD extra, sem particionar ou formatar o HD.

    É uma maneira alternativa de instalar o Batocera, muito mais prática, fácil
e sem risco de ferrar com tudo.

    O procedimento consiste em configurar o gerenciador de inicialização "GRUB"
para dar boot na imagem (.img) do Batocera em quaquer pasta no HD.

Testado com:
    Batocera.PLUS 2.14 (PC 64 Bits)
    Batocera.Linux 31 (PC 64 Bits)
    Batocera.Linux 30 (PC 32 Bits)
    Partições ext4, ntfs, fat32
    Sistema de particionamento MBR e GPT

Como instalar o BATOCERA-HD-EDITION?
--------------------------------------------------------------------------------
    Descompacte a imagem do Batocera (Batocera.PLUS-x.xx-stable.7z) com o
programa de sua preferência, recomendo que use o 7zip https://www.7-zip.org

    Crie uma pasta ou sub-pasta na unidade raiz (C:), sem usar espaços e
caracteres especiais para evitar aborrecimentos futuros e descompacte a imagem
no diretório criado.

Exemplo Windows:
    C:\batocera\Batocera.PLUS.img

Exemplo Linux:
    apt-get update
    sudo apt-get install p7zip
    sudo mkdir -p /opt/batocera
    sudo 7zr x Batocera.PLUS-2.14-dev.7z -o/opt/batocera/Batocera.PLUS.img

    Windows: Se você usa somente o Windows você vai precisar baixar um programa
para inicializar o Batocera, baixe o Grub2Win gratuitamente no seguinte site
https://sourceforge.net/projects/grub2win o programa também está na imagem do
Batocera.PLUS na pasta tools.

    Depois de baixar instale o programa em suas configurações padrões.

    Este programa vai adicionar uma entra extra no menu de inicialização do
Windows, o programa não altera em nada a inicialização do Windows apenas
adiciona uma opção extra para poder inicializar outros sistemas operacionais.
Se encontrar problemas na instalação, desative o securboot pela bios do PC.

    Ao término da instalação do programa será aberto um utilitário que permite
personalizar o menu de inicialização do Windows, não customize nada, apenas
conclua a instalação.

    Windows: O Grub2Win por padrão será instalado direto na unidade C:\grub2 e
suas configurações ficam no arquivo C:\grub2\grub.cfg para adicionar sistemas
operacionais na inicialização do Windows basta editar este arquivo usando o
programa o Notepad++ disponível em https://notepad-plus-plus.org/downloads.
IMPORTANTE, NÃO edite o arquivo grub.cfg usando o bloco de notas do windows e
nem o Microsoft Word.

    Linux: Se você usa Linux, parabéns! Você é um cara esperto ;-) e não precisa
de mais nada, pois o Linux normalmente já vem com o Grub2 instalado por padrão
e com bons editores de texto.

    Linux + Windows: Na imagem do Batocera.PLUS, na pasta batocera-hd-edition
tem o arquivo grub.cfg pré-configurado para iniciar o Batocera.PLUS e o
Batocera.Linux, este mesmo arquivo também está disponível no seguinte link:
https://github.com/AlexxandreFS/Batocera.PLUS/blob/master/boot/batocera-hd-edition/grub.cfg

    O arquivo grub.cfg precisa ser editado de acordo com o local onde você
descompactou a imagem do batocera, procure pela seguinte linha:

    set BATOCERA_PLUS=/LiveLinux/Batocera.PLUS.img

    Modifique de acordo com o local onde você descompacou a imagem, lembre-se de
respeitar as letras maiúsculas e minúsculas.

    Para rodar o Batocera.Linux será necessário baixar um arquivo extra
"batocera-hd-edition.gz" o Batocera.PLUS não precisa deste arquivo e você pode
ter as duas versões do batocera no mesmo computador.
O arquivo pode ser encontrado na imagem do Batocera.PLUS na pasta
batocera-hd-edition ou pode ser baixado deste link
https://raw.githubusercontent.com/AlexxandreFS/Batocera.PLUS/master/boot/batocera-hd-edition/batocera-hd-edition.gz

    Procure pelas seguintes linhas e modifique conforme a localiazação dos
arquivos no seu computador, remova também o # do início das linhas.

    #set BATOCERA_LINUX=/LiveLinux/Batocera.Linux.img
    #set BATOCERA_HD_EDITION=/LiveLinux/batocera-hd-edition.gz

    Windows: Copie o arquivo modificado para a pasta de instalação do Grub2Win
C:\grub2\grub.cfg e mande substituir por este arquivos que acabamos de editar.
Se você seguiu exatamente o que foi explicado, basta reiniciar o computador e
escolher o Grub2Win no menu de inicialização e se divertir.

    Linux: Debian e derivados, remova as seguintes linhas do arquivo grub.cfg

    set default="0"
    set timeout="0"
    
    Renomei o arquivo grub.cfg para custom.cfg, copie o arquivo para a pasta
/boot/grub

    sudo cp grub.cfg /boot/grub/custom.cfg

Atualize o grub para reconhecer o Batocera no nemu de inicialização.

    sudo update-grub

    Linux Outros: Remova as seguintes linhas do arquivo grub.cfg

    set default="0"
    set timeout="0"

Adicione as demais linhas do arquivo grub.cfg no final do arquivo
/boot/grub/grub.cfg

    sudo cat grub.cfg >> /boot/grub/grub.cfg


    Windows + Linux: A imagem do batocera servirá apenas para carregar o sistema
básico não é possivel adicionar jogos, temas etc dentro da imagem do batocera,
nem mesmo as as configurações do sistema como idioma são salvas na imagem, assim
que o sistema for reiniciado tudo será perdido!
    Para poder salvar as suas configurações é preciso procurar uma opção no menu
do Batocera referente ao local de armazenamento será mostrado todas as partições
de todos os HD instalados no computador e você deve escolher um deles para
salvar as modificações. O batocera não altera a partição ele apenas cria uma
pasta com o nome de batocera e usa esta pasta para salvar as configurações, roms
temas, etc... Esta pasta pode ser acessada diretamete pelo Windows ou Linux.
    
Conclusões
--------------------------------------------------------------------------------
Embora possa parece complicado o processo de instalação é bem simples.
Criei esta modificação para poder instalar o batocera em meu PC sem modificar as
minhas configurações e para não ficar somente em meu computador resolvi
compartilhar.

RECEITA DE BOLO PARA USUÁRIOS UBUNTU E DERIVADOS

Abra um terminal e faça login como root

    sudo su [Digite sua senha]

Instale o 7zip para poder descompactar o arquivo.
    
    apt-get update
    apt-get install p7zip

Acesse a pasta onde você baixou o Batocera.PLUS pelo terminal.
    
    cd Downloads

Descompacte a imagem do Batocera.PLUS.
    
    7zr x Batocera.PLUS-x.x-dev.7z

Crie uma pasta para colocar o Batocera.PLUS
    
    mkdir -p /opt/batocera

Mova a imagem descompactada para a pasta que você acaba de criar.

    mv  Batocera.PLUS.img /opt/batocera

Edite o arquivo grub.cfg que você baixou com um editor de texto de sua preferência.
    
    nano grub.cfg
    
    ou

    gedit grub.cfg

Configure o local para onde você descompactou a imagem do Batocera.

    set BATOCERA_PLUS=/opt/batocera/Batocera.PLUS.img

Remova estas duas linhas ou comente com # no início da linha.
Não se esqueça de salvar o arquivo.

    #set default="0"
    #set timeout="0"

Mova o arquivo grub.cfg para o diretório de boot do Grub.
    
    mv grub.cfg /boot/grub/custom.cfg

O Ubuntu por padrão não mostra o menu de inicialização do Grub para poder
escolher qual sistema você quer inicializar (Ubuntu ou Batocera)
Para ter acesso a este menu você tem duas opções:

1 - Ligar o computador segurando a tecla Shift da esquerda até o menu aparecer.

2 - Editar o arquivo de preferências do Grub para o menu sempre aparecer.
    
    nano /etc/default/grub
    
    ou
    
    gedit /etc/default/grub

Procure pela seguinte linha e remova ou comente com um # no início da linha.

    #GRUB_HIDDEN_TIMEOUT=0
    
Aproveite que você já está aqui e aumente o tempo para poder selecionar qual
sistema você pretende carregar. Neste caso 60 segundos = 1 Minuto

    GRUB_TIMEOUT=60

Por fim atualize as configurações do Grub e reinicie o PC

    update-grub
